About

This site provides a Json API to fetch the MCC/MNC codes

All requests to the JSON API should include application/json in the Accept HTTP Header

API calls returning an error, a non 2xx HTTP status code, will also return a JSON object on the format: 

                
{
    "errormessage":"Entry missing"
}

List all MCC/MNC Codes

URL /e212api.v1/e212/

Parameters

None

Method

GET

Success HTTP Status code

200

Example Response

GET Request for URL /e212api.v1/e212/ returns an arry of MCC/MNC entries

                
[
{
    "code": {
        "mcc": "412",
        "mnc": "01"
    },
    "country": "Afghanistan",
    "operator": "AWCC"
},
{
    "code": {
        "mcc": "412",
        "mnc": "20"
    },
    "country": "Afghanistan",
    "operator": "Roshan"
}
]

List all MCC/MNC Codes with a given MCC

URL /e212api.v1/e212/:mcc

Parameters

:mcc The MCC code to list

Success HTTP Status code

200

Error HTTP Status code

404 when the given :mcc is not found

Example Response

GET Request for URL /e212api.v1/e212/412 returns an arry of MCC/MNC entries

                
    [
    {
        "code": {
            "mcc": "412",
            "mnc": "01"
        },
        "country": "Afghanistan",
        "operator": "AWCC"
    },
    {
        "code": {
            "mcc": "412",
            "mnc": "20"
        },
        "country": "Afghanistan",
        "operator": "Roshan"
    }
    ]

GET Request for URL /e212api.v1/e212/12345 returns a 404 error

                
{
    "errormessage":"Entry missing"
}

Get MCC/MNC entry for the given MCC/MNC

URL /e212api.v1/e212/:mcc/:mnc

Parameters

:mcc The MCC code to get

:mcc The MNC code to get

Success HTTP Status code

200

Error HTTP Status code

404 when the given :mcc and :mnc is not found

Example Response

GET Request for URL /e212api.v1/e212/412/01 returns an MCC/MNC entry

                
    {
        "code": {
            "mcc": "412",
            "mnc": "01"
        },
        "country": "Afghanistan",
        "operator": "AWCC"
    }

GET Request for URL /e212api.v1/e212/242/999 returns a 404 error

                
{
    "errormessage":"Entry missing"
}
